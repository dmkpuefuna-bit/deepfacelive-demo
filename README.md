# Daniel's DeepFaceLive Demo (Simplified)

A lightweight web demo inspired by [DeepFaceLive](https://github.com/iperov/DeepFaceLive) for real-time AI face swapping.  
Built with *Python* and *Gradio*, this demo simulates facial blending online without heavy GPU requirements.
